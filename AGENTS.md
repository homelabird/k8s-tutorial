# AGENTS.md

## Remote Safety

- Never push to the upstream/original repository (`sailor-sh/CK-X`).
- Never open a pull request against the upstream/original repository.
- Do not push to any remote or create any pull request unless the user explicitly asks in the current conversation.
- Default to local-only changes. If remote publication is ever requested, use the user's fork only after explicit confirmation.
- Do not update or remove these remote-safety rules unless the user explicitly asks for that change.
